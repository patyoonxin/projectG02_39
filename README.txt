Installation manual
1. Open the Xampp link given in INSTALLER folder and download the latest version of Xampp server.
2. After download, open the Xampp control panel and start the Apache and MySql.
3. Open the phpMyAdmin.
4. Create a new database name opd.
5. Import he database dump file given in SYSTEM folder.
6. Done.
